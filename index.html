Application Décentralisée (DApp) pour l'Affichage des Notes sur Blockchain

Projet de Fin d'Études Licence - Université Moulay Ismail, Errachidia

Conception et développement d'une application décentralisée (DApp) innovante utilisant la technologie blockchain Ethereum pour sécuriser l'affichage et la gestion des notes académiques. Cette solution élimine les risques de fraude et garantit la transparence des données grâce à l'immuabilité de la blockchain.

  • Développement de smart contracts en Solidity pour la gestion des notes et réclamations
  • Intégration d'IPFS (InterPlanetary File System) pour le stockage décentralisé des données
  • Interface utilisateur développée avec React.js et Web3.js
  • Authentification sécurisée via MetaMask et gestion des transactions Ethereum
  • Mise en œuvre d'un système de réclamations décentralisé pour les étudiants
  • Tests et déploiement sur blockchain Ganache pour l'environnement de développement

Blockchain Solidity React.js Web3.js IPFS Ethereum Smart Contracts Ganache MetaMask

2025

Application de Télésurveillance Médicale

Projet Fin de Module - Université Ibn Zohr, Ouarzazate

Conception et développement d’une application de télésurveillance des patients basée sur une architecture multi-agents (SMA) utilisant la plateforme JADE. Le système surveille en temps réel la fréquence cardiaque d’un patient, détecte automatiquement les anomalies (tachycardie, bradycardie) et notifie un professionnel de santé via un agent de notification. Une interface graphique en Java Swing permet de piloter le système (démarrage/arrêt des agents) et de suivre les journaux en temps réel. Le projet démontre la coopération entre agents autonomes pour la gestion distribuée de données médicales.

Java JADE Multi-Agent System Java Swing
